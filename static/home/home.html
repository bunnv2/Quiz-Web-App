Quiz - projekt zaliczeniowy

Jest to projekt zaliczeniowy, który został wykonany w ramach przedmiotu "Programowanie obiektowe i graficzne" oraz "Bazy danych". Aplikacja została napisana w języku Python w oparciu o framework Django i bazę danych sqlite3. Do wystylowania strony został wykorzystany framework Bootstrap 5. Autorami projektu są:
Karol Ciepły, Alicja Gruca, Bartosz Jarzyński, Wojciech Królik.
